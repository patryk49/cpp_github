Allocator "A" is any type that works with thoose procedures:	
	Memblock alloc(A, size_t)                     - allocate memory with default alignment
	void free(A, Memblock)                        - free specific memory (not always possible)

optional allocator procedures:
	Memblock alloc(A, size_t, size_t)             - allocate memory with specific alignment
	void free(A)                                  - free all memory
	Memblock realloc(A, Memblock, size_t)         - reallocate memory
	Memblock realloc(A, Memblock, size_t, size_t) - reallocate memory
